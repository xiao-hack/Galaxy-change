## 🚀 安装指引

插件下载：[Download](https://github.com/xiao-hack/Galaxy-change/releases)

插件安装：`Extensions -> Add -> Select File -> Next`

自行构建： `build.gradle -> shadowJar`（gradlew shadowJar）

**注意事项**:

- [首次使用必读](https://github.com/xiao-hack/Galaxy-chage/wiki)
- Burp版本不低于 `v2023.10.3.7`
